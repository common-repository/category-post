Add "New Post" by Categories in admin menu.

== Description ==
This plugin adds "Add New" by Categories in admin menu.




== Installation ==

1. Download the category-post.zip file to your computer.
2. Unzip the file.
3. Upload the `category-post` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
